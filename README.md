# 🍰 Bakery App Assignment

A simple Android application designed to showcase bakery products and beverages. Built with **Java** and **Android SDK**, this app provides a user-friendly interface for browsing categories like Birthday Cakes, Beverages, and Cupcakes.

## 📱 Features
- **Category Selection:** Visual grid layout with icons for different product categories.
- **Custom UI:** 
  - Black-themed Action Bar with centered title.
  - Custom icons for cakes, beverages, and bakery items.
- **Responsive Layout:** Uses `ConstraintLayout` for adaptive positioning on various screen sizes.
- **Edge-to-Edge Support:** Implements modern Android window insets for a clean look on newer devices.

## 🛠️ Tech Stack
- **Language:** Java
- **Build System:** Gradle (Kotlin DSL: `build.gradle.kts`)
- **UI Components:** ConstraintLayout, Toolbar, ImageView, TextView
- **Theme:** Material Design (AppCompat)
